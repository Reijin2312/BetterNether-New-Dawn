## About

**BetterNether: New Dawn** is an independently maintained, unofficial continuation of BetterNether for Fabric, Quilt, and NeoForge.

The project continues development of the original BetterNether experience with maintenance, bug fixes, compatibility improvements, balance adjustments, quality-of-life changes, and support for newer Minecraft versions and mod loaders.

BetterNether transforms the Nether with new biomes, plants, mobs, structures, materials, recipes, tools, food, building blocks, visual effects, and expanded world generation.

The mod is designed for both client and server environments.

> [!IMPORTANT]
> BetterNether: New Dawn is not maintained or endorsed by the original BetterNether developers.
>
> Please do not report New Dawn-specific issues to the upstream BetterNether repository or the official BetterX support channels. Use the [New Dawn issue tracker](https://github.com/Reijin2312/BetterNether-New-Dawn/issues) instead.

## Features

* Dozens of new plants for the Nether
* New Nether mobs, including challenging creatures such as the Naga
* New unique Nether biomes
* Vertical and volumetric biome generation
* Compatibility with other biome mods and datapacks
* New materials, recipes, food, tools, weapons, and items
* Farmable resources through plant and mob farms
* New Nether structures
* Large and rare Nether Cities
* Dark dungeons occupied by Nether creatures
* Unique building blocks and additional variants of vanilla blocks
* Configurable blocks, items, structures, biomes, and plant density
* Config GUI support through Mod Menu
* Integrations with other mods

## New Dawn improvements

BetterNether: New Dawn aims to remain close to the original project while providing shared fixes, selected balance changes, and loader-specific compatibility improvements.

Current improvements include:

* NeoForge support for modern Minecraft versions
* Continued Fabric and Quilt maintenance
* TerraBlender compatibility
* Compatibility with mods using TerraBlender
* Improved music system
* Improved Ruby compatibility tags
* Fixes for block drops, mob drops, world generation, and structures
* Fixes for the Respawn Statue and Nether Brewing Stand
* Balance and quality-of-life adjustments
* Improved translations
* Better consistency between supported mod loaders

## Download and installation

Download BetterNether: New Dawn from the official CurseForge page:

### [Download from CurseForge](https://www.curseforge.com/minecraft/mc-mods/betternether-neoforge/files)

1. Install a supported Minecraft version.
2. Install the appropriate NeoForge, Fabric, or Quilt loader.
3. Download the BetterNether: New Dawn file matching your loader and Minecraft version.
4. Install the required dependencies.
5. Place the downloaded `.jar` files in your Minecraft `mods` directory.
6. Launch the game.

Make sure the Minecraft version, mod loader, BetterNether version, and dependency versions match.

---

## Required dependencies

BetterNether: New Dawn uses libraries from the New Dawn ecosystem.

Requirements may vary between Minecraft versions and mod loaders. Always check the relations and dependencies shown on the selected CurseForge file.

* [BCLib: New Dawn](https://www.curseforge.com/minecraft/mc-mods/bclib-neoforge)
* [WorldWeaver: New Dawn](https://www.curseforge.com/minecraft/mc-mods/worldweaver-neoforge)
* [WunderLib: New Dawn](https://www.curseforge.com/minecraft/mc-mods/wunderlib-neoforge)

The Fabric and Quilt editions also require the appropriate version of Fabric API.

## Development setup

Clone the repository:

```bash
git clone https://github.com/Reijin2312/BetterNether-New-Dawn.git
cd BetterNether-New-Dawn
```

Select the branch for the version and loader you want to work on:

```bash
# NeoForge 1.21.1
git checkout master

# Fabric / Quilt 1.21.1
git checkout fabric-1.21.1

# NeoForge 1.21.11
git checkout port/1.21.11

# NeoForge 26.1.x
git checkout 26.1
```

Edit `gradle.properties` if necessary.

Open `build.gradle` as a new Gradle project in IntelliJ IDEA and allow Gradle to import and configure the project.

---

## Local BetterX dependencies

By default, the project can use local versions of other BetterX dependencies from sibling directories.

For example, a local BCLib checkout can be placed next to this repository:

```text
projects/
├── BetterNether-New-Dawn/
└── BCLib/
```

Local dependency inclusion can be configured in:

```text
settings.betterx.gradle
```

When local dependency sources are not enabled, the project uses configured binary dependencies instead.

## Building

Run:

```bash
./gradlew build
```

On Windows:

```bat
gradlew.bat build
```

The compiled mod files will be available in:

```text
build/libs
```

## Reporting issues

Before opening an issue:

1. Make sure you are using the latest available version.
2. Verify that you downloaded the correct file for your Minecraft version and loader.
3. Check that all required dependencies are installed.
4. Confirm that the dependency versions match your BetterNether version.
5. Test without unrelated mods when possible.
6. Include the latest log or crash report.
7. Include a complete mod list.
8. Provide clear steps to reproduce the problem.

Report bugs through the official New Dawn issue tracker:

### [Open an issue](https://github.com/Reijin2312/BetterNether-New-Dawn/issues)

Do not report New Dawn-specific bugs to the original BetterNether developers.
